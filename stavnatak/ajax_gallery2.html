Co jsem si měl zjistit a co jsem vlastně zjistil

  • AJAX
    + request a response
  • API
  • HTTP status codes

Request = dotaz na server

Request je např. již při prvním načtení stránky, kdy se serveru zeptám "Hej, chci otevřít tuhle stránku!" Nebo "Potřebuju tuhle informaci".

Response = odpověď serveru

Response je zase, kdy mi server řekne "Jasně, stáhni si tohle a tohle".

AJAX = slouží pro vytváření interaktivní webů.

Jednou z hlavních předností je, že zabraňuje neustálemu reloadování stránek, protože si získává info ze serveru na pozadí. Dále se přes něho můžeme průběžně ptát serveru například, zda není zadávaný nick již obsazený, anebo vytvořit našeptávač. Může sloužit i pro přihlašování, kde kontroluje zda souhlasí jméno a heslo.
