API = databáze informací

Soubory api jsou pro nás databází, ze které si taháme informace. Například pomocí JSONu.

GET, POST, DELETE, PUT

Jsou metody získávání informací, kde přes GET sbírám informace, přes POST je vypisuji, DELETE mažu a PUT aktualizuji.

Příklady aplikací API

Google maps

Pokud si dám vyhledat restaurace v místě hledání, tak si mapy sáhnou do api databáze a vypíší mi je nad mapou (z mapy se stane vrstva)

Aplikace na počasí

Opět mi podle zadaných údajů vypíše aktuální předpověď, kterou si vytáhne z databáze, aniž bych musel neustále reloadovat web.

Přihlášení přes facebook, google apod.

Kde si zavoláme na server, zdali je již přihlášený k dané platformě, případně jej vyzveme aby se přihlásil, kde se nám správnost údajů ověřuje přímo na serveru platformy kam mi vysíláme request.

Facebook, twitter apod.

Stránky načítají nový obsah, aniž bych musel reloadovat web stránku, tím že vyšlou request na server a response zařadí do správné sekce.

Booking

Zde si posíláme například requesty na server, jestli je zvolený termín volno a server nám odpovídá zda ano či ne a dle toho vypíšem odpověď.

Spotify, Netflix apod

Přístupy do API se využívají ať už při vyhledávání, anebo při načítání obsahu, kde se mění jen požadované sektory.
